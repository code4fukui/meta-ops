# xmldom-es

> English README: [README.md](README.md)


> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

純粋なJavaScript W3C標準に基づく (XML DOM Level 2 Core) DOMParser および XMLSerializer モジュール。

**Live demo**: https://code4fukui.github.io/xmldom-es/xmldom.js

**Version**: 0.8.2

## テクノロジー ステック

**Node.js / NPM Packages:**

- `@stryker-mutator/core@5.6.1`
- `auto-changelog@2.4.0`
- `eslint@8.17.0`
- `eslint-config-prettier@8.5.0`
- `eslint-plugin-es5@1.5.0`

## プロジェクト構造

- `lib/` — 共有ライブラリとユーティリティ

## インストール & セットアップ

### 前提条件

- Node.js (v16 or higher)
- Npm

### 手順

1. リポジトリをcloneする

   ```bash
   git clone https://github.com/code4fukui/xmldom-es.git
   cd xmldom-es
   ```

2. 依存関係をインストール

   ```bash
   npm install
   ```

3. プロジェクトを実行

   生産用サーバーを開始:
   ```bash
   npm start
   ```

## 利用可能なコマンド

- `npm run lint` — eslint lib test
- `npm run changelog` — auto-changelog --unreleased-only
