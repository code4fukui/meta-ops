## インストール & セットアップ

### 必要条件

- Node.js (v16 またはそれ以上)
- Npm

### ステップ

1. リポジトリをクローン

   ```bash
   git clone https://github.com/code4fukui/xmldom-es.git
   cd xmldom-es
   ```

2. 依存パッケージをインストール

   ```bash
   npm install
   ```

3. プロジェクトを実行

   プロダクションサーバーを開始:
   ```bash
   npm start
   ```


## 利用可能なコマンド

- `npm run lint` — eslint lib test
- `npm run changelog` — auto-changelog --unreleased-only

- `npm run stryker:dry-run` — stryker run -m '' --reporters progress
